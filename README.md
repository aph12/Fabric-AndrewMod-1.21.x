# Minecraft Fabric Mod

Java-based Minecraft Fabric mod adding custom tools, armor, blocks, and items using object-oriented design and modular systems.

# Features
- Custom tool and armor materials with balanced durability and damage
- Modular item and block registration using enums and utility classes
- JSON-based recipes, loot tables, and tags
- Structured project organization using Java packages

# Technologies
Java, Fabric API, Git, Object-Oriented Programming
